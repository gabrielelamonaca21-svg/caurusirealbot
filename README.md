# caurusirealbot

Bot Discord di base creato per avviare velocemente il progetto.

## Installazione

1. Apri il terminale nella cartella del progetto
2. Esegui `npm install`
3. Copia `.env.example` in `.env` e inserisci il token del bot Discord

## Avvio

`npm start`

## Struttura

- `src/index.js` - avvia il bot e carica comandi/eventi
- `src/commands/` - comandi bot
- `src/events/` - eventi Discord
- `src/config/` - file di configurazione
- `src/utils/` - utilità e loader opzionali
- `src/data/` - storage locale dei dati
